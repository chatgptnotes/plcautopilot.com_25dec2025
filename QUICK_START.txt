STEP 1: Create PLC Program
────────────────────────────
Command: python program_motor_startstop.py

What it does:
• Opens EcoStruxure Machine Expert Basic
• Creates "MotorStartStop" project
• Selects TM221CE24T PLC
• Programs ladder logic (Start/Stop with seal-in)
• Compiles program
• Saves project

STEP 2: Download to PLC
────────────────────────
Command: python auto_download_plc.py

Prerequisites:
• PLC powered ON
• USB cable connected
• EcoStruxure open with project

What it does:
• Opens download dialog
• Selects USB connection
• Detects PLC
• Downloads program
• Switches PLC to RUN mode

STEP 3: Wire the Hardware
──────────────────────────
Read: WIRING_AND_TESTING_GUIDE.md

Connections:
• Input %I0.0  ← Start Button (NO)
• Input %I0.1  ← Stop Button (NC)
• Output %Q0.0 → Motor Contactor

STEP 4: Test the System
────────────────────────
Read: WIRING_AND_TESTING_GUIDE.md (Testing section)

Tests:
• Press Start → Motor runs
• Release Start → Motor stays on (seal-in)
• Press Stop → Motor stops

 PROGRAM DETAILS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PLC Model: Schneider Electric TM221CE24T

Inputs:
  %I0.0 = Start Button (Normally Open)
  %I0.1 = Stop Button (Normally Closed)

Outputs:
  %Q0.0 = Motor Contactor

Logic:
  ┌─────────────────────────────────┐
  │ Start  Stop         Motor       │
  │ ──┤ ├────┤/├────────( )──      │
  │                      │          │
  │ Motor Seal-in        │          │
  │ ──┤ ├────────────────┘          │
  └─────────────────────────────────┘

Operation:
  1. Press START → Motor energizes
  2. Motor self-holds via seal-in contact
  3. Press STOP → Motor de-energizes

Software won't open:
  • Verify software is installed
  • Try different search term
  • Run: python open_ecostruxure.py

Motor won't start:
  • Check Stop button (%I0.1 should be ON)
  • Verify PLC in RUN mode
  • Check wiring connections

Motor won't stop:
  • Test Stop button
  • Check %I0.1 input status
  • Verify NC contact wiring

PLC won't download:
  • Check USB cable
  • Verify PLC powered ON
  • Try manual download first
